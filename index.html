The Solution

How llm-d Fixes Each Wall

⇄

EPP Smart Routing

Routes requests to the optimal pod, minimizing cross-node communication. Reduces east-west traffic by 60%.

🗃

Prefix-Aware Scheduling

Directs requests to pods with KV cache locality. Cache hit rate climbs from 8% to 85%+.

🎯

Cache-Aware Routing

Routes to pods with the prompt prefix already cached. Redundant prefill drops by 10x.

☰

Staged Rollout Controller

Eliminates thundering herd. Pods load in sequence with health-gate sequencing.

🗂

InferencePool CRD

Batches endpoint management, reducing etcd writes by 10x. Keeps control plane healthy.

🛡

Full Stack Integration

All layers together remove cascade preconditions. Production-ready at 128 GPUs.
